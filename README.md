# Dark Premium Portfolio (GitHub Pages)

A modern, clean, trendy one-page portfolio built with **only HTML/CSS/JS**.
No frameworks. No external services. Easy GitHub Pages deploy.

## Files
- `index.html`
- `styles.css`
- `app.js`

## Customize
Open `app.js` and edit:
- `PROFILE` (name, email, socials)
- `PROJECTS` (add/remove projects, tags, links)

Thatâ€™s it. The Projects section auto-renders from the `PROJECTS` array, including search + tag filters.
